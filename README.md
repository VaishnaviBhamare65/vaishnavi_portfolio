# Vaishnavi Portfolio

React + Vite + Tailwind CSS portfolio site.

## Deployment

**Vercel:**
1. Push to [https://github.com/VaishnaviBhamare65/vaishnavi_portfolio.git](https://github.com/VaishnaviBhamare65/vaishnavi_portfolio.git)
2. Vercel auto-deploys from main branch.

**Local Development:**
```bash
npm install
npm run dev
```

**Build & Preview:**
```bash
npm run build
npm run preview
```
